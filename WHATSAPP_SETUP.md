# 📱 Настройка WhatsApp API для Orbit Digital

Вы можете выбрать один из двух способов отправки заявок в WhatsApp:

---

## ✅ Способ 1: Twilio (Рекомендуется для бизнеса)

### Что это даст:
- ✅ Профессиональный сервис
- ✅ Надежная доставка сообщений
- ✅ Поддержка медиа (изображения, документы)
- ⚠️ Платный (примерно $0.01-0.03 за сообщение)

### Шаг 1: Создать аккаунт Twilio

1. Откройте https://www.twilio.com/
2. Нажмите "Sign Up"
3. Заполните форму и подтвердите email
4. Завершите верификацию номера телефона

### Шаг 2: Получить учетные данные

1. Откройте https://console.twilio.com/
2. На главной странице вы увидите:
   - **Account SID** - скопируйте это значение
   - **Auth Token** - скопируйте это значение

Пример:
```
Account SID:  ACxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
Auth Token:   xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
```

### Шаг 3: Включить WhatsApp Sandbox

1. В левом меню перейдите: **Messaging → Try it out → Send an SMS**
2. Затем перейдите: **Messaging → WhatsApp → Sandbox**
3. Следуйте инструкциям:
   - Отправьте SMS с текстом "join" на указанный номер Twilio
   - Вы получите подтверждение
4. **Скопируйте номер WhatsApp "From"** (это номер бота Twilio)

Пример: `whatsapp:+1415xxxxxxx`

### Шаг 4: Получить номер для получения сообщений

Это может быть:
- Ваш личный номер WhatsApp
- Номер, зарегистрированный в Twilio

Формат: `whatsapp:+79991234567` (вместо +7999 укажите ваш код страны и номер)

### Шаг 5: Заполнить .env.local

Откройте `.env.local` в корне проекта и заполните:

```env
# Twilio
VITE_TWILIO_ACCOUNT_SID=ACxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
VITE_TWILIO_AUTH_TOKEN=xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
VITE_TWILIO_WHATSAPP_FROM=whatsapp:+1415xxxxxxx
VITE_WHATSAPP_PHONE_TO=whatsapp:+79991234567
```

## 💰 Способ 2: Make.com (Бесплатный/Дешевый для начинающих)

### Что это даст:
- ✅ Бесплатно (до 1000 операций в месяц)
- ✅ Простая настройка
- ✅ Не требует платежа
- ⚠️ Может быть медленнее
- ⚠️ Требует отдельной настройки сценария

### Шаг 1: Создать аккаунт Make.com

1. Откройте https://make.com/
2. Нажмите "Sign up free"
3. Заполните форму и подтвердите email

### Шаг 2: Создать сценарий для WhatsApp

1. Нажмите "Create a new scenario"
2. Выберите "Webhooks" как триггер (первый модуль)
3. Нажмите "Add" и скопируйте **Webhook URL**

Пример:
```
https://hook.make.com/abc123def456ghi789jkl012mno345
```

### Шаг 3: Добавить модуль WhatsApp

1. Нажмите "+" и найдите "WhatsApp Business" (или ваш WhatsApp сервис)
2. Подключите свой аккаунт WhatsApp
3. Настройте отправку сообщения с данными из webhook

**Или используйте HTTP модуль** для отправки на ваш сервер:

1. Добавьте модуль "HTTP"
2. Выберите метод POST
3. Укажите URL вашего сервера (если у вас есть свой бэкэнд)

### Шаг 4: Заполнить .env.local

```env
VITE_MAKE_WEBHOOK_URL=https://hook.make.com/abc123def456ghi789jkl012mno345
```

### Шаг 5: Активировать сценарий

1. Нажмите кнопку "Turn on" внизу экрана
2. Нажмите "Save"

## 🐛 Решение проблем

### "Message not sent to WhatsApp"

**Для Twilio:**
- Проверьте что Account SID и Auth Token правильные
- Убедитесь что номера в формате: `whatsapp:+1234567890`
- Проверьте консоль браузера (F12) → Network → смотрите ответ API

**Для Make.com:**
- Убедитесь что webhook активирован (кнопка "Turn on")
- Проверьте что Webhook URL правильный
- Запустите сценарий вручную для тестирования

### "CORS error"

Это значит что браузер блокирует запрос. Решение:
- Для Twilio: используйте Twilio SDK (может быть добавлен в проект)
- Для Make.com: работает через proxy, должно работать

### "Credentials not configured"

- Проверьте что `.env.local` существует
- Проверьте что переменные названы правильно
- **Перезагрузите сервер:** Ctrl+C и `npm run dev` заново
