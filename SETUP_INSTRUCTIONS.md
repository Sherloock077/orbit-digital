# 📋 Инструкция по настройке отправки заявок в Telegram и WhatsApp

## ✅ Что было сделано

Ваша форма "Оставить заявку" теперь может отправлять данные в:
- ✅ **Telegram** (бесплатно)
- ✅ **WhatsApp** (платно, через Twilio или другой сервис)

---

## 🤖 Настройка Telegram Bot (ОБЯЗАТЕЛЬНО)

### Шаг 1: Создать бота в Telegram

1. Откройте Telegram и найдите: **@BotFather**
2. Отправьте команду: `/start`
3. Затем: `/newbot`
4. Следуйте инструкциям:
   - Придумайте имя для бота (например: "Orbit Digital Bot")
   - Придумайте username для бота (например: "@orbit_digital_bot")
5. **Скопируйте токен бота** - он вам понадобится (выглядит как: `123456789:ABCdefGHIjklmnoPQRstuvwxyzABCdefGHI`)

### Шаг 2: Узнать ID чата

1. Добавьте своего бота в группу или напишите ему в личные сообщения
2. Отправьте сообщение боту
3. Откройте в браузере: `https://api.telegram.org/bot<ВАШЕ_ТОКЕН>/getUpdates`
4. Замените `<ВАШЕ_ТОКЕН>` на токен из Шага 1
5. Найдите `"chat":{"id": ЧИСЛО}` - это ваш Chat ID

### Шаг 3: Добавить данные в .env.local

Откройте файл `.env.local` в корне проекта и заполните:

```env
VITE_TELEGRAM_BOT_TOKEN=123456789:ABCdefGHIjklmnoPQRstuvwxyzABCdefGHI
VITE_TELEGRAM_CHAT_ID=123456789
```

**Сохраните файл** (Ctrl+S)

## 📱 Настройка WhatsApp (ОПЦИОНАЛЬНО)

### Вариант 1: Twilio (Платный, самый популярный)

1. Зарегистрируйтесь на https://www.twilio.com/
2. Получите API ключ и номер WhatsApp
3. Добавьте в `.env.local`:

```env
VITE_WHATSAPP_API_KEY=your_twilio_api_key
VITE_WHATSAPP_PHONE=+1234567890
```

### Вариант 2: Make.com (Зумирование, более простой)

1. Откройте https://make.com/
2. Создайте сценарий (scenario) для отправки в WhatsApp
3. Получите Webhook URL
4. Обновите в `src/services/messageService.js` функцию `sendWhatsAppMessage`:

```javascript
export const sendWhatsAppMessage = async (formData) => {
  const webhookUrl = 'https://your-make-webhook-url';
  // ... отправка на webhook
};
```

### Вариант 3: WhatsApp Business API (От Meta)

Документация: https://developers.facebook.com/docs/whatsapp

## 📊 Как работает форма теперь

1. **Пользователь заполняет форму** с именем, email, телефоном и сообщением
2. **При отправке:**
   - Данные проверяются (обязательные поля: имя, email, сообщение)
   - Отправляются в Telegram через Telegram Bot API
   - Отправляются в WhatsApp (если настроено)
   - Показывается сообщение об успехе
   - Форма очищается

## 🐛 Решение проблем

### Сообщение не приходит в Telegram

1. Проверьте токен в `.env.local` - скопируйте точно без пробелов
2. Проверьте Chat ID
3. Откройте консоль браузера (F12) и посмотрите ошибки

### "Telegram credentials not configured"

- Убедитесь, что файл `.env.local` существует в корне проекта
- Проверьте что переменные названы правильно: `VITE_TELEGRAM_BOT_TOKEN` и `VITE_TELEGRAM_CHAT_ID`
- Перезагрузите сервер разработки: остановите (Ctrl+C) и запустите заново (`npm run dev`)

### WhatsApp не работает

- Сначала убедитесь что Telegram работает
- WhatsApp требует настройки Twilio или другого сервиса
